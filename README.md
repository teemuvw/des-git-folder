# DES Encryption Python Library

Python implementation of Data Encryption Standard (DES) algorithm, padding support for key and plaintext, automated testing using pytest

## Features

- DES encryption of 64-bit plaintext and 64-bit key
- ASCII input used
- One 64 bit block maximum (eight characters) for input & key
- Minimum one character input & key
- 16 subkeys generated
- Matrix according to https://csrc.nist.gov/files/pubs/fips/46-3/final/docs/fips46-3.pdf
- Hexadecimal output
- pytest included
